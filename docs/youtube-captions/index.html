Download a caption file from any video, drop it on Vertano, and get back a clean, correctly timed subtitle track — plus translated versions ready to re-upload. Nothing leaves your machine.

The doubled-line problem, solved.

YouTube auto-generated captions are "rolling" captions: every cue repeats the previous completed line before adding new words. Download the .srt or .vtt and most lines appear twice, with timing that overlaps itself. Vertano recognizes that exact structure and rebuilds one clean line per moment, correctly timed.

What Vertano does with a caption file

  • Removes the rolling duplication. Doubled lines collapse into a single clean caption, with gaps and timing preserved.
  • Keeps everything timed. The cleaned track stays a valid subtitle file you can load into any player or re-upload to YouTube.
  • Translates on device. Pick target languages and Vertano writes one correctly timed .srt or .vtt per language, translated locally.
  • Leaves the original alone. Your downloaded file is never modified; cleaned and translated tracks are saved alongside it.
  • Handles both formats. SubRip .srt and WebVTT .vtt, in and out.

Manually authored captions pass through untouched

Only the rolling auto-caption pattern is rewritten. Hand-authored subtitle files and karaoke-style tracks are recognized and left exactly as they are, so nothing you crafted by hand gets flattened.

Honest about translation coverage

On the Mac app, translation uses Apple's on-device language packs. A handful of languages Apple does not yet cover, such as Urdu, Bengali, Persian, Punjabi, and Pashto, will not translate, but the cleaned source-language track is still produced. Audio in those languages still transcribes and translates to English through the transcription engine itself.
